import { Aside, Badge } from '@astrojs/starlight/components';

Player options are passed via the `options` key of the constructor's config object.

```js
new Vlitejs('#player', {
  options: {
    // Options here
  }
});
```

---

## controls

```ts
type controls = boolean;
```

**Media:** <Badge text="Video only" variant="note" />

**Default:** `true`

Display the control bar of the player. When set to `false`, all controls are hidden.

```js
new Vlitejs('#player', {
  options: {
    controls: false
  }
});
```

<Aside type="note">
  Vimeo provider requires a Plus or Pro account to use this feature (see [Vimeo
  plans](https://vimeo.zendesk.com/hc/en-us/articles/228068968-Compare-Vimeo-plans)).
</Aside>

## autoplay

```ts
type autoplay = boolean;
```

**Default:** `false`

Enable autoplay when the player loads. Due to browser autoplay policies, enabling this option automatically activates [`muted`](#muted).

```js
new Vlitejs('#player', {
  options: {
    autoplay: true
  }
});
```

<Aside type="note">
  See [Autoplay policy changes](https://developer.chrome.com/blog/autoplay) for more information.
</Aside>

## autoHide

```ts
type autoHide = boolean;
```

**Media:** <Badge text="Video only" variant="note" />

**Default:** `false`

Automatically hide the control bar after a period of inactivity. The controls reappear on mouse move or touch.

```js
new Vlitejs('#player', {
  options: {
    autoHide: true
  }
});
```

## autoHideDelay

```ts
type autoHideDelay = number; // milliseconds
```

**Media:** <Badge text="Video only" variant="note" />

**Default:** `3000`

Delay in milliseconds before the control bar hides when [`autoHide`](#autohide) is enabled.

```js
new Vlitejs('#player', {
  options: {
    autoHideDelay: 5000
  }
});
```

---

## providerParams

```ts
type providerParams = Record<string, unknown>;
```

**Default:** `{}`

Override the embed parameters sent to the provider's API or SDK.

```js
new Vlitejs('#player', {
  options: {
    providerParams: {
      // YouTube example: disable related videos
      rel: 0
    }
  }
});
```

<Aside type="note">
  See [YouTube embed options](https://developers.google.com/youtube/player_parameters#Parameters)
  and [Vimeo embed options](https://github.com/vimeo/player.js/#embed-options) for available
  parameters.
</Aside>
