**`vlitejs`** is a fast and lightweight JavaScript library for customizing video and audio players. Written in native JavaScript — no dependency. Built around a Provider API (HTML5, YouTube, Vimeo, Dailymotion) and a Plugin API (subtitle, Picture-in-Picture, Cast, AirPlay, IMA…).

**6 KB gzip** — the lightest player in its class.

## Install

```bash
npm install vlitejs
```
